CKAN(Comprehensive Knowledge Archive Network)

CKAN은 데이터 허브 및 데이터 포털을 위한 오픈 소스 DMS (데이터 관리 시스템)입니다.

List of CKAN Extensions
이름 기능 참고
DataStore CKAN DataStore 확장은 CKAN 자원에서 구조화 된 데이터를 저장하기위한 임시 데이터베이스, 리소스 파일에서 데이터를 가져와서 DataStore에 저장할 수 있음. Datastore 공식 문서
DataPusher 자동적으로 CKAN instance의 리소스를 Datastore에 저장하도록 도움을 주는 standalone web service DataPusher 공식 문서
Havest ckan 확장을위한 data-sharing 프레임 워크를 제공하고 수확 소스 및 작업을 관리하기 위해 CLI 및 WUI를 CKAN에 추가하여 관리함. Harvest github
